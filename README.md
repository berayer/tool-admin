# tool-admin

工具集合后台管理系统,将所有工具集成在一起,方便管理

## 项目模块

| 模块名称              | 功能作用                  |
|-------------------|-----------------------|
| tool-admin-common | 通用模块,封装基本功能和共享实体类     |
| tool-admin-main   | 服务启动类, 核心功能实现服务       |
| tool-admin-imes   | 提供联思数据服务,通过RPC调用,内部服务 |

## 项目依赖

| 依赖名称     | 功能作用     |
|----------|----------|
| sa-token | 提供权限校验功能 |
